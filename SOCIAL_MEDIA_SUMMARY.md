## üîß Troubleshooting

### Common Issues

**"Login not detected"**
- Script waits for auto-detection but may timeout
- Log in manually when browser opens
- Script will continue after timeout

**"Could not find element"**
- Social media UIs change frequently
- Manual fallback always available
- Content copied to clipboard for manual paste

**"No unread messages"**
- May not have unread messages
- Check browser window for verification
- Script shows what it can detect
